# JML Evidence Checklist (Template)

## Purpose
Ensure Joiner–Mover–Leaver events have required approvals, logs, and evidence retained for audit readiness.

## Joiner (New Hire)
- [ ] Request ticket created (who/what/when)
- [ ] Manager approval captured
- [ ] Access provisioned based on role/RBAC
- [ ] MFA/SSO enrollment completed (if required)
- [ ] Evidence retained (screenshots/logs/export)

## Mover (Role Change)
- [ ] Role change request ticket created
- [ ] Manager + system owner approvals captured
- [ ] Access removed from old role
- [ ] Access granted for new role
- [ ] Privileged access reviewed (if applicable)
- [ ] Evidence retained

## Leaver (Termination/Offboarding)
- [ ] HR/manager termination notice captured
- [ ] Deprovisioning completed within SLA
- [ ] Privileged/admin access removed first (if applicable)
- [ ] Sessions/tokens revoked (if applicable)
- [ ] Device access disabled (if applicable)
- [ ] Evidence retained

## Evidence Artifacts to Retain (examples)
- [ ] Ticket ID(s)
- [ ] Approval email/attestation
- [ ] Export showing access state before/after
- [ ] Admin console screenshots (as needed)
- [ ] System logs (auth/provisioning/deprovisioning)
- [ ] Exception documentation (if applicable)
