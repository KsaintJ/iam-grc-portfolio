# Joiner–Mover–Leaver (JML) Lifecycle Controls

This project documents identity lifecycle management controls designed to reduce access risk and support compliance requirements.

## Objective
Define standardized Joiner, Mover, and Leaver controls that ensure timely provisioning, modification, and removal of access based on role changes.

## Scope
- New hire access provisioning (Joiner)
- Role changes and access updates (Mover)
- Timely deprovisioning and termination handling (Leaver)
- Approval, logging, and evidence retention

## Deliverables
- JML workflow diagram (high-level)
- Control statements for each lifecycle stage
- Evidence checklist (tickets, approvals, logs)
- Deprovisioning SLAs and accountability model

## Key Concepts
IAM • Identity Lifecycle • JML • Access Governance • Risk Reduction • Audit Readiness
